基于Java的springboot和vue结合,js,css,mybatis
功能：
客户预定房间->跳转支付宝支付
酒店管理员显示已支付，房间自动锁定，前台不再显示该房间，等顾客到现场办理入住，退房后房间打开。
顾客可以根据订单号对酒店进行评论
